# MorningBooster 도움말

## 설정 방법

1.  **[설정]** 버튼을 클릭하여 설정 창을 엽니다.

2.  **Chrome 경로**: 사용하시는 크롬 브라우저의 실행 파일(chrome.exe) 경로를 정확히 입력해주세요.
    (보통 `C:\Program Files\Google\Chrome\Application\chrome.exe` 입니다.)

3.  **창 그룹 관리**:
    *   `[+ 새 창 추가]` 버튼으로 새로운 브라우저 창 그룹을 만들 수 있습니다. (최대 4개)
    *   각 창 그룹의 `[X 창 삭제]` 버튼으로 해당 그룹을 삭제할 수 있습니다.

4.  **링크 관리**:
    *   각 창 그룹의 `[+ 링크 추가]` 버튼으로 해당 창에서 열고 싶은 웹사이트 주소를 추가할 수 있습니다. (창당 최대 4개)
    *   각 링크 옆의 `[X]` 버튼으로 해당 링크를 삭제할 수 있습니다.

5.  **로컬 프로그램 관리**:
    *   카카오톡, 디스코드 등 실행하고 싶은 프로그램의 전체 경로를 입력하세요.
    *   `[+ 프로그램 추가]`로 새 프로그램을, `[X]`로 기존 프로그램을 삭제할 수 있습니다.

6.  모든 설정이 완료되면 **[저장 후 닫기]** 버튼을 눌러 변경사항을 저장하세요.

## 실행 방법

*   설정이 완료된 후 메인 화면의 **[실행]** 버튼을 누르면 저장된 모든 창과 프로그램이 한 번에 실행됩니다.
